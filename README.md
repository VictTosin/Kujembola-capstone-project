# Insurance Claim Prediction - Capstone Project

This project focuses on predicting the probability that a building will file an insurance claim during the Insured period using supervised machine learning techniques.

#Problem Statement
To predict whether a building will make atleast one insurance claim during the insured period. The target variable is Claim.
